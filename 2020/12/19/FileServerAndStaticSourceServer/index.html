golang的静态资源服务器与文件服务器

静态资源服务器

静态资源一般指服务器发送到客户端的文件，常见的比如js、css、图片、音频等

在golang中实现一个静态资源服务器十分简单

  1. 第一个参数”/files/“，代表静态资源请求的URL

  2. func StripPrefix(prefix string, h Handler) Handler

    这个函数的官方解释如下

    StripPrefix returns a handler that serves HTTP requests by removing the given prefix from the request URL’s Path and invoking the handler h. StripPrefix handles a request for a path that doesn’t begin with prefix by replying with an HTTP 404 not found error.

简单来说就是将请求的URL去掉前缀，并执行所给的handler

  1. func FileServer(root FileSystem) Handler

    FileServer returns a handler that serves HTTP requests with the contents of the file system rooted at root.

    会以root为根路径，处理访问本地”/tmp”文件夹的HTTP请求

  2. http.Dir
    这实际上是一个类型转换，http.Dir(“/tmp”)是利用本地tmp目录实现一个文件系统
